# ECommerce Automation Framework

This is a Java + Selenium WebDriver test automation framework using TestNG and Maven.

## Tech Stack
- Java
- Selenium WebDriver
- TestNG
- Maven
- WebDriverManager

## How to Run
1. Open in IntelliJ or Eclipse
2. Run `mvn clean test`

## Test Cases
- Valid login with email and password

## Site Under Test
